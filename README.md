# 자동차 경주 게임🚗
## 구현해야 할 기능

### 경주할 자동차 이름 입력

- 콤마(,) 를 기준으로 구분한다.
#### 예외상황
- 입력한 자동차 이름이 5자가 넘는경우
- 자동차 이름 맨앞이 공백인 경우
### 진행할 경기 회수 입력

#### 예외 상황

- 숫자가 아닌 문자나 기호등을 입력한 경우
- 예외 상황 에러문구는 `[ERROR]`로 시작

### 게임 진행

- 각 자동차 별로 0~9의 랜덤 값 생성
- 랜덤 값이 `4` 이상이면 전진한다.
- 랜덤 값이 `3` 이하이면 멈춘다.
- 입력한 경기 회수만큼 위 과정을 반복한다.

### 우승자 안내

- 단독 우승일 경우 자동차 이름을 출력
- 공동 우승일 경우 콤마(,)와 공백 으로 자동차 이름을 구분하여 출력
